1.**Hangman Game🎮**
   
A simple text-based Hangman game built with Python.

2.**About:**

This project was built as part of the **CodeAlpha Python Internship**.
The game picks a random word and the player guesses it letter by letter.

3.**Features**

- 5 predefined words
- 6 incorrect guesses allowed
- Tracks guessed letters
- Win and lose conditions
  
4.**Concepts Used**
  
- `random` module
- `while` loop
- `if-else` statements
- Strings and Lists

5. **How to Run**


1. Make sure Python is installed on your system
2. Download `Hangman Game Code` file
3. Open terminal/command prompt
4. Run this command:

```python Hangman Game Code```

5. Start guessing letters and enjoy the game! 🎮
