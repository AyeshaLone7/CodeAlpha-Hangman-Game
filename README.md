1. Hangman Game
A simple text-based Hangman game built with Python.
2.About:
This project was built as part of the **CodeAlpha Python Internship**.
The game picks a random word and the player guesses it letter by letter.
3.Features
- 5 predefined words
- 6 incorrect guesses allowed
- Tracks guessed letters
- Win and lose conditions
4.Concepts Used
- `random` module
- `while` loop
- `if-else` statements
- Strings and Lists
